# Jandig MR
## Jandig Tutorial – Quick Guide to Test
Logging in and opening the exhibit
1. Open the Jandig app.​
2. Log in
   - create an user at **https://dev.jandig.app/users/signup/**
3. In the main menu, go to **Remix Exhibit**.​
4. Select **Remix MR Exhibit** to see all available mixed reality exhibitions.​
5. Choose the exhibition called **CheckAll**.​
6. Choose a name for your narrative.

# Editing the objects
1. Wait for the exhibition to load and for the objects to appear.
2. Point your hand at an object; a white line from the controller to the object shows what object is selected.​
3. Use the pinch gesture to grab and move the objects to the desired orientation and positions​
4. At any point, you can **Save**.
5. When finished, use the menu option Save and Exit to store your changes and exit.​
# Viewing the finished exhibition
1. Go back to the main menu and select See Local Exhibit to open the just created narrative..
2. Enjoy!
